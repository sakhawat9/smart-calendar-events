Smart calendar events plugin for wordPress.

== Description ==

Experience seamless event management with the Smart Calendar Events plugin, your ultimate solution for creating, organizing, and showcasing events effortlessly. With a robust set of features tailored to enhance your event management experience, Smart Calendar Events sets itself apart by offering intuitive event creation tools, a dynamic calendar table display, and seamless integration with your website's front end. You can show events anywhere on your website through the [smart-calendar-events] shortcode.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `smart-calendar-events.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Plugin features:

- Intuitive Event Creation: Easily create events with a user-friendly interface that allows you to input event details such as title, excerpt, date, image, and description.
- Dynamic Calendar Table Display: Showcase your events in a visually appealing and organized manner using a dynamic calendar table display. Users can quickly browse through events by date and access event details with ease.
- Responsive Design: Ensure a seamless user experience across devices with a responsive design that adapts to various screen sizes, including desktops, tablets, and smartphones.
- Integration with Website: The [smart-calendar-events] shortcode is a powerful tool for integrating event listings and details into your website's front end. Here's a breakdown of the attributes you can use with this shortcode:
  1. post-per-page: This attribute specifies the number of events to display per page. For example, if you set post-per-page="6", the shortcode will display 6 events per page. You can adjust this number based on your preferences and layout requirements.
  2. order: This attribute determines the order in which events are displayed. For example, if you set order="ASC/DESC"
- Multilingual Support: Reach a global audience with multilingual support that allows you to translate event details and interface elements into multiple languages.
