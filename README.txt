Smart calendar events plugin for wordPress.

== Description ==

The Smart Calendar Events plugin should allow creating events, displaying them in a calendar table, and showing events in the front end. Designed to seamlessly enhance your event management experience, Smart Calendar Events offers a range of powerful features and tools that empower you to take your event showcasing to the next level.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `smart-calendar-events.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Plugin features:

- Add new user information.
- Edit the existing information.
- Delete existing user information.
- View all information in admin dashboard.
- Sortable admin dashboard by ID user name, and user email.
- All text strings used in the plugin are translatable to support internationalization.
